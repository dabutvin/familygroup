# familygroup

The Isaac and Celia Sklar Family Group — a family history site with diary entries, a family tree, and photo gallery.

Built with Vue 3, Vuetify 3, and Vite.

## Project setup
```
npm install
```

### Development server
```
npm run dev
```

### Production build
```
npm run build
```

### Preview production build
```
npm run preview
```

### Lint
```
npm run lint
```
